# 8-Puzzle Solver Using Branch and Bound (C)

## Overview
This project implements an 8-Puzzle solver using the Branch and Bound (A*) search algorithm in C. It finds the optimal sequence of moves from a given initial state to the goal state using Manhattan distance heuristic.

## Features
- Solves 8-Puzzle problem (3x3 sliding puzzle)
- Uses Branch and Bound (A*) search strategy
- Manhattan distance heuristic for optimal path finding
- Detects solvability of input state
- Prints step-by-step solution path

## Algorithm Used
- Branch and Bound (A* Search)
- Heuristic: Manhattan Distance
- State-space search with visited state tracking

## Tech Stack
- Language: C
- Concepts: Data Structures, Algorithms, Heuristic Search
